# Видео инструкции

## Как добавить видео в приложение

1. **Поместите видео файлы в эту папку** (`react-app/public/videos/`)
2. **Используйте форматы**: `.mp4` (рекомендуется), `.webm`, `.ogg`
3. **Оптимизируйте видео**:
   - Разрешение: 1920x1080 или меньше
   - Битрейт: 2-5 Mbps для HD контента
   - Используйте H.264 кодек для .mp4

## Пример добавления видео в Step8.jsx

```javascript
const videos = [
  {
    src: "/videos/my-instruction-video.mp4", // ваш файл в этой папке
    title: "Название видео",
    description: "Описание видео инструкции",
    poster: "/images/video-poster.jpg" // постер в папке images
  }
];
```

## Рекомендуемые постеры

Создайте изображения постеров размером 1920x1080px и поместите их в папку `react-app/public/images/`.

## Поддержка браузеров

- Chrome: все форматы
- Firefox: все форматы  
- Safari: .mp4 (H.264)
- Edge: все форматы

## Troubleshooting

Если видео не загружается:
1. Проверьте путь к файлу
2. Убедитесь что файл существует
3. Проверьте формат видео
4. Проверьте права доступа к файлу
